# PRD — 머니하우 (money-how)

> 수입과 지출의 흐름을 한눈에 시각화하는 개인 재무 대시보드

---

## 1. 개요

### 배경

개인 재무 관리 앱은 많지만, 대부분 숫자 나열이나 파이차트에 그친다. 사람들이 실제로 궁금한 것은 **"내 돈이 어디서 와서, 어디로 흘러가는가"** 이다. 이 흐름을 직관적으로 보여주는 도구가 없다.

### 목표

Sankey 다이어그램을 활용해 수입 → 지출의 돈 흐름을 시각적으로 표현하고, 사용자가 직접 데이터를 입력해 자신만의 재무 흐름도를 만들 수 있는 웹 도구를 제공한다.

### 타겟 사용자

- 자신의 월 수입/지출 구조를 한 장으로 파악하고 싶은 개인
- 가계부를 쓰고 있지만 전체 구조가 잘 안 보이는 사람
- 재무 목표(저축률, 고정비 비율 등)를 시각적으로 점검하고 싶은 사람

## 2. 핵심 가치 제안

| 기존 도구 | 머니하우 |
|---|---|
| 숫자/표로 나열 | 흐름으로 시각화 |
| 앱 설치 필요 | 웹 브라우저만으로 즉시 사용 |
| 복잡한 입력 과정 | 항목명 + 금액만 입력 |
| 데이터 저장 필수 | 저장 없이 즉시 확인 가능 |

---

## 3. 기능 요구사항

### 3.1 핵심 기능 (MVP)

#### F-01. Sankey 다이어그램 시각화

- 왼쪽: 수입 항목 (초록색 계열)
- 가운데: 총계 노드
- 오른쪽: 지출 항목 (다색 구분)
- 각 흐름의 두께는 금액 비율에 비례
- 수입 > 지출 시 **잔여** 항목 자동 생성
- 지출 > 수입 시 **부족분** 항목 자동 표시
- 마우스 호버 시 해당 노드와 연결된 흐름 하이라이트
- 컨테이너 크기에 따라 자동 리사이즈 (반응형)

#### F-02. 데이터 편집

- 수입 항목 추가 / 수정 / 삭제
- 지출 항목 추가 / 수정 / 삭제
- 차트 제목 수정 (예: "2024년 3월 가계부")
- 금액 단위 수정 (만원, 억원, 달러 등)
- 수정 즉시 차트에 실시간 반영

#### F-03. 요약 지표 표시

- 총 수입 합계
- 총 지출 합계
- 잔여 또는 부족 금액 (색상으로 구분)

### 3.2 이후 기능 (Post-MVP)

#### F-04. 공유 / 저장

- URL 쿼리 파라미터 또는 해시로 데이터 인코딩하여 공유 링크 생성
- 이미지(PNG/SVG)로 내보내기
- 로컬스토리지 자동 저장 (마지막 입력 데이터 유지)

#### F-05. 다중 기간 관리

- 월별 데이터 여러 개 저장 및 탭 전환
- 기간 비교 뷰 (나란히 두 달 비교)

#### F-06. 템플릿

- 직장인 기본 / 프리랜서 / 사업자 등 카테고리 프리셋 제공

#### F-07. 분석 지표

- 저축률 자동 계산 및 권고 기준(20%) 대비 표시
- 고정비 / 변동비 구분 태그
- 카테고리별 전월 대비 증감

| 항목 | 요구사항 |

## 6. 데이터 모델

```ts
interface Category {
  id: string;
  name: string;   // 항목명 (예: "급여", "식비")
  amount: number; // 금액 (단위 생략, 별도 unit 필드로 관리)
}

interface FinancialData {
  title: string;     // 차트 제목
  unit: string;      // 금액 단위 (예: "만원")
  incomes: Category[];
  expenses: Category[];
}
```

## 7. 사용자 여정

```
[방문]
  │
  ▼
[샘플 데이터로 차트 확인]   ← 첫 방문 시 월간 가계부 샘플 로드
  │
  ▼
["데이터 편집" 버튼 클릭]
  │
  ▼
[수입/지출 항목 수정]        ← 이름, 금액 입력 / 항목 추가·삭제
  │
  ▼
[차트 실시간 업데이트]       ← 키 입력 즉시 반영
  │
  ▼
[결과 공유 / 저장]           ← (Post-MVP)
```

## 8. 성공 지표

| 지표 | 목표 |
|---|---|
| 첫 방문 → 데이터 편집 전환율 | 60% 이상 |
| 평균 세션 시간 | 3분 이상 |
| 차트 이미지 저장/공유 | DAU 대비 20% (Post-MVP) |
